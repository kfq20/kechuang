# 答辩材料：碳小侠未来环保实验室

## 一、3 分钟答辩稿

各位老师、评委好，我的作品叫《碳小侠未来环保实验室》。

我设计这个作品，是因为我发现大家都知道要保护环境，但是很多同学并不清楚，自己每天到底可以做哪些环保行动。所以我想做一个既好玩、又有科技感的作品，让同学们在互动中学习环保知识。

我的作品是一个互动网页，主要有四个部分。

第一部分是环保数据看板。它可以把环保行动变成看得见的数据，比如低碳指数、环保徽章和一周行动趋势。这样同学们会更有参与感，也更容易理解“环保是可以坚持和记录的”。

第二部分是环保分类挑战赛。我设计了适合小学生的题目，比如矿泉水瓶应该放进哪个垃圾桶、离开教室前怎么做更节能。答题以后会马上反馈结果，还会根据答题情况升级称号，让学习变得像闯关游戏一样有趣。

第三部分是家庭低碳计划生成器。输入名字，选择想改善的方向，比如省电、垃圾分类或者绿色出行，系统就会自动生成一份适合小学生执行的计划。这样环保不只停留在网页里，还能延伸到家庭生活中。

第四部分是趣味展示功能，比如今日环保任务抽卡和未来城市宣言，可以让现场展示更活泼，也能增强作品的完整性。

我的作品创新点主要有三点。第一，它不是普通的海报，而是一个可以互动的数字作品。第二，它把知识学习和生活实践结合起来，让环保真正落地。第三，它的界面比较生动，适合在比赛中现场演示。

如果以后继续改进，我还想加入班级排行榜、更多题库和真实调查数据，让这个作品变成一个真正可以在校园中推广的小平台。

我的答辩结束，谢谢大家。

## 四、评委可能提问与参考回答

### 1. 你为什么选择环保这个主题？
因为环保和我们的生活关系最密切，小学生每天都能接触到垃圾分类、节约用电和绿色出行，所以这个主题既有意义，也容易实践。

### 2. 你的作品和普通海报有什么不同？
普通海报主要是展示信息，我的作品可以互动。用户可以答题、生成计划、抽任务，参与感更强，也更有科技感。

### 3. 你觉得这个作品最大的亮点是什么？
我觉得最大的亮点是把环保知识和实际行动结合起来，不只是告诉大家“要环保”，而是帮助大家知道“今天可以怎么做”。

### 4. 如果继续完善，你还想加什么？
我想加入班级排行榜、更多题目、调查问卷和真实数据统计，让它变成班级同学共同参与的平台。

### 5. 这个作品适合在哪些地方使用？
适合科技节、环保主题班会、校园展示活动和小学生科创比赛。
