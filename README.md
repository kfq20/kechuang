# 碳小侠未来环保实验室

这是一个面向小学生的互动式网页作品，主题聚焦“低碳环保 + 科技体验 + 生活实践”。

## 项目亮点

- 主题贴近校园和家庭生活，容易理解，也容易坚持实践。
- 界面色彩明快，带有未来实验室风格，适合手机和电脑浏览。
- 作品包含可互动功能，不只是静态海报：
  - 环保分类闯关（独立大题库）
  - 一周环保数据看板
  - 家庭低碳计划生成器
  - 今日环保任务抽卡
  - 未来城市宣言切换
- 所有内容均为中文，打开就能直接体验。

## 运行方式

这是一个纯前端静态项目。为了正常加载外置题库 `data/questions.json`，建议使用本地服务器或部署后访问。

如果希望本地演示更稳定，可以在当前目录启动一个简单服务器：

```bash
python3 -m http.server 8080
```

然后在浏览器打开：

```bash
http://localhost:8080
```

## 文件说明

- `index.html`：项目主页面
- `styles.css`：界面样式与动画效果
- `script.js`：互动功能逻辑与题库加载
- `data/questions.json`：环保大题库，目前包含 92 道题
- `docs/项目说明.md`：提交时可参考的项目介绍
- `docs/答辩材料.md`：介绍稿、PPT 提纲、常见问答

## 适合的使用场景

- 小学生环保主题学习
- 信息素养与网页创作练习
- 校园科技节互动体验
- 综合实践或班级主题活动
